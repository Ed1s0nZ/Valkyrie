# Valkyrie

## 为什么有这个项目？
- 逛GitHub的时候偶然间发现一个不错的[项目](https://github.com/Maka8ka/NGLite)，此项目是一款基于区块链网络的匿名跨平台远控程序，无需任何公网资源即可上线。
- 在认真阅读作者写的工具原理以及区块链平台官方文档后，个人对此项目较为感兴趣。
- 在实际使用此工具时，被其优秀的免杀性能，以及隐蔽性深深吸引。所以想尝试在此原理的基础上(站在巨人的肩膀上)，写一款自己趁手的工具。
